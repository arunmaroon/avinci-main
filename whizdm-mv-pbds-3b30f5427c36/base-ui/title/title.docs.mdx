A component that renders a Title

### Component usage

### Render title!

```js
<ThemeProvider>
  <Title title="Hello" />
</ThemeProvider>
```

### Render title with description!

```js live
<ThemeProvider>
  <Title variant="screen" title="Some title" subtitle="This is the short description" />
</ThemeProvider>
```
