import { pothosTokens } from ".";

A component that does something special and renders text in a div.

### Render opacity!

```js live
<>
  {Object.entries(pothosTokens.opacity).map(([opacityName, opacityValue]) => (
    <div>
      <div style={{ backgroundColor: "black", color: "white", padding: "1rem", opacity: opacityValue }}>{opacityName}</div>
    </div>
  ))}
</>
```
